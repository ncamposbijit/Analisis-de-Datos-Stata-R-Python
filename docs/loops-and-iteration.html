10 Loops and Iteration

10.1 For Loops

For loops iterate over a sequence of values:

# Basic for loop
for (i in 1:5) {
  print(paste("Iteration", i))
}
## [1] "Iteration 1"
## [1] "Iteration 2"
## [1] "Iteration 3"
## [1] "Iteration 4"
## [1] "Iteration 5"
# Iterating over vector elements
fruits <- c("apple", "banana", "cherry")

for (fruit in fruits) {
  print(paste("I like", fruit))
}
## [1] "I like apple"
## [1] "I like banana"
## [1] "I like cherry"
# Building results in a loop
squares <- numeric(10)
for (i in 1:10) {
  squares[i] <- i^2
}
squares
##  [1]   1   4   9  16  25  36  49  64  81 100

10.2 While Loops

While loops continue until a condition becomes FALSE:

# Basic while loop
counter <- 1

while (counter <= 5) {
  print(paste("Counter is", counter))
  counter <- counter + 1
}
## [1] "Counter is 1"
## [1] "Counter is 2"
## [1] "Counter is 3"
## [1] "Counter is 4"
## [1] "Counter is 5"
# Practical example: finding first value above threshold
x <- 1
while (2^x < 1000) {
  x <- x + 1
}
print(paste("2^", x, "=", 2^x, "is the first power of 2 above 1000"))
## [1] "2^ 10 = 1024 is the first power of 2 above 1000"
